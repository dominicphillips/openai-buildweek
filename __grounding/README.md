# Grounding index

This folder is the shared source of truth for product context, research, decisions, and inspiration. Keep it concise enough that an agent can re-enter the project quickly, and link to primary sources instead of copying them wholesale.

- `PRODUCT.md` — audience, promise, journey, and first vertical slice
- `DESIGN_PRINCIPLES.md` — responsible creative principles and interaction rules
- `BRANDS.md` — initial preference-selector catalog and neutral trait tags
- `ARCHITECTURE.md` — system boundaries, ports, state, and integration decisions
- `SOUND.md` — optional creative-focus audio direction
- `SOURCES.md` — external source trail and what each source supports

Add screenshots, supplied references, and licensed inspiration in clearly named subfolders only after confirming they are safe to commit. Raw uploads and generated media belong in ignored runtime folders, not here.
